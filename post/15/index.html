정리

함수형 프로그래밍이란?

수학 함수를 사용하고, 부수 효과를 피하는 것이 특징인 프로그래밍 패러다임, 부수 효과가 없는 함수는 순수 함수라고 불림

// 순수 함수
function sum(list){
    let result = 0;
    list.forEach((elem) => {
        result += elem;
    })
    return result
}

// 순수 함수가 아님
